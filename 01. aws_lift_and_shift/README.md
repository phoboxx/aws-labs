## Components

- **Load Balancer**: Distributes incoming traffic
- **Tomcat Server**: Application server
- **Mariadb**: Backend data storage
- **Memcached**: In-memory caching layer
- **RabbitMQ**: Message broker for async processing

## Getting Started

1. Clone the repository
2. Configure your infrastructure parameters
3. Deploy the stack
4. Access the application through the load balancer

## Requirements

- AWS Account
- Appropriate IAM permissions
- Required AWS resources provisioned
